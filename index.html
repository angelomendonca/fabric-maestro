Maestro is a Fabric based toolkit for managing systems. There is also integration with cloud providers (currently Amazon EC2 and Rackspace) via Apache Libcloud.

Installation

  • pip install maestro

Usage

Either download the included fabfile or create your own:

from maestro.core import *
from maestro import system as sys

You can then run fab -l to see the available tasks.
